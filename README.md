# Cleopatra
Cleopatra is an intuitive iPhone app to optimize a salon environment. It features an interactive countdown timer.
# Technologies Used
* Xcode 13
* SwiftUI 
* Swift
# Features
* Interactive Countdown Timer
* Background Timer Support
